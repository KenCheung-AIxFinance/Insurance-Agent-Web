# 🧾《[Agent.AI](http://Agent.AI) 智能保險助手》PRD v2.0（繁體中文）

## 一、產品概述

**產品定位：**
為香港及大灣區的保險經紀人（AIA、保誠等）提供 **智能條款查詢與建議書生成工具**。
本系統不是銷售導向，而是專注於**提升專業效率與解釋精準度**。

**產品形式：**

* Streamlit 應用（MVP 階段）
* 單人帳號授權制（年費）
* 使用 DeepSeek + LangChain 技術架構

| 目標 | 說明 |

## 四、核心功能模組（MVP）

### 1️⃣ 條款檢索與人性化解釋

**輸入：**

* 上傳保單條款 PDF
* 使用自然語言提問，如「癌症是否包含早期診斷？」

**AI 輸出內容：**

* **專業層解釋：** 精準引用條文內容，附來源頁碼
* **簡化解釋：** 面向客戶的口語化說明（AI自動生成）
* **引用追蹤：** 條文標註與相似段落來源展示

**技術要點：**

* DeepSeek API (LangChain Wrapper)
* PDF Parser（PyPDF2 + LangChain DocumentLoader）
* 向量資料庫：FAISS / Chroma
* RAG 查詢（Retriever + Contextual Compression）


---

### 2️⃣ Proposal 智能轉換模組

**目標：**
將不同公司提供的 Proposal（通常格式各異）轉換為標準化、易讀、可分享的「簡化建議書」。

**流程：**


1. 拖曳上傳 PDF / Excel Proposal 檔案
2. 系統自動識別結構（產品名稱、保障範圍、金額、期限）
3. 應用預設模板轉換為「簡化建議書」樣式
4. 使用者可編輯、導出 PDF

**模板特色：**

* 可根據不同公司自訂樣式（顏色、字體、結構）
* 自動生成建議摘要（AI 將保單內容轉成「一段式總結 + 三點亮點」）
* 可添加經紀個人簽名 / 聯絡資料

**技術要點：**

* 表格與文字結構抽取：pdfplumber + pandas
* LangChain Text-to-Template 填充
* Streamlit 編輯區預覽 + 一鍵導出 PDF
