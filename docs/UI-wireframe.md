# 7️⃣ 錯誤狀態與系統提示 UI

### ❌ 搜尋無結果

> 「未在已上傳文件中找到相關條款。建議上傳更多條款文件，或更改搜尋關鍵字。」

### ❌ PDF 無法 OCR

> 「文件解析失敗，請重新上傳更清晰的版本。」

### 📄 OCR 進行中

> Loading spinner：「正在處理 PDF（OCR + Chunking），平均需 5–15 秒」


---

# 📦 8️⃣ 流程 Wireflow（動線）

### 條款檢索動線（RAG）

```
輸入問題 → 搜尋 → 顯示條款原文 → 顯示專業解釋 → 顯示客戶版 → （可選）生成 PDF
```

### Proposal → 建議書動線

```
上傳 Proposal → 選模板 → 生成 → 預覽 → 下載 PDF / Word
```

### 文件管理動線

```
文件列表 → 查看文件 → OCR / Chunk → 條款檢索可引用
```
