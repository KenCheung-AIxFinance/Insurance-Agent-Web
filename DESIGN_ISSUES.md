# 设计不一致性问题清单

## 概述
本文档记录了 InsurAgent 应用中发现的视觉和交互不一致性问题，为设计统一化提供改进方向。

## 1. 视觉不一致问题

### 1.1 色彩使用不一致

#### 问题描述
- **首页快速动作按钮**使用了硬编码颜色值：
  - `bg-blue-600 hover:bg-blue-700`
  - `bg-green-600 hover:bg-green-700` 
  - `bg-purple-600 hover:bg-purple-700`
- 这些颜色未遵循设计系统的语义色彩规范

#### 影响范围
- `/src/pages/Home/index.tsx` - 快速动作按钮

#### 建议解决方案
- 使用设计系统定义的语义色彩变量
- 统一为 `variant="default"` 或创建新的语义变体

### 1.2 渐变背景不一致

#### 问题描述
- **首页快速动作区**使用了自定义渐变背景：
  - `bg-gradient-to-r from-blue-50 to-indigo-50`
- 该渐变未在设计系统中定义

#### 影响范围
- `/src/pages/Home/index.tsx` - 快速动作区域

#### 建议解决方案
- 在设计中系统定义标准渐变
- 或使用现有的背景色变量

### 1.3 文本颜色不一致

#### 问题描述
- **首页**中使用了硬编码文本颜色：
  - `text-slate-800`
  - `text-slate-600`
- **教学文档页面**中同样使用了硬编码颜色

#### 影响范围
- `/src/pages/Home/index.tsx`
- `/src/pages/Tutorials/index.tsx`

#### 建议解决方案
- 统一使用 CSS 变量：`text-foreground`、`text-muted-foreground`

## 2. 交互不一致问题

### 2.1 按钮变体使用不一致

#### 问题描述
- **导航按钮**使用 `variant="ghost"`
- **快速动作按钮**使用硬编码颜色
- **教学文档按钮**使用默认变体

#### 影响范围
- 整个应用中的按钮交互状态

#### 建议解决方案
- 统一按钮变体的使用场景
- 建立明确的按钮使用指南

### 2.2 悬停状态不一致

#### 问题描述
- 自定义按钮的悬停状态透明度不一致
- 导航链接的悬停状态使用透明度变化

#### 影响范围
- 所有交互元素

#### 建议解决方案
- 统一悬停状态的透明度变化规范

## 3. 布局不一致问题

### 3.1 间距系统不一致

#### 问题描述
- 不同页面使用了不同的间距值
- 缺少统一的间距比例系统

#### 影响范围
- 所有页面和组件

#### 建议解决方案
- 建立标准的间距比例系统
- 使用 Tailwind 的间距工具类

### 3.2 容器宽度不一致

#### 问题描述
- 布局组件使用 `max-w-6xl`
- 教学文档页面使用 `container mx-auto`

#### 影响范围
- 页面布局容器

#### 建议解决方案
- 统一容器宽度定义
- 在设计中系统明确容器规范

## 4. 组件库不一致问题

### 4.1 组件导入路径不一致

#### 问题描述
- 部分组件从 `@/components/general/ui/` 导入
- 部分组件从 `@/components/ui/` 导入

#### 影响范围
- 所有使用组件的文件

#### 建议解决方案
- 统一组件导入路径
- 建立清晰的组件目录结构

### 4.2 组件变体定义不一致

#### 问题描述
- 按钮组件定义了额外的 `success` 和 `warning` 变体
- 这些变体未在其他组件中保持一致

#### 影响范围
- 组件库的扩展性

#### 建议解决方案
- 统一语义变体的定义和使用

## 5. 优先级排序

### 高优先级
1. 色彩使用不一致 - 影响品牌一致性
2. 按钮变体使用不一致 - 影响用户体验

### 中优先级  
3. 布局不一致 - 影响视觉统一性
4. 组件导入路径不一致 - 影响开发效率

### 低优先级
5. 渐变背景不一致 - 视觉影响较小
6. 悬停状态不一致 - 细节优化

## 6. 改进时间线

### 第一阶段（立即执行）
- 修复首页快速动作按钮的色彩问题
- 统一文本颜色使用

### 第二阶段（近期）
- 统一按钮变体使用规范
- 建立标准的间距系统

### 第三阶段（长期）
- 重构组件导入路径
- 完善组件变体系统
